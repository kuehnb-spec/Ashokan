What Ashokan is

Ashokan lets you edit HTML files the way a word processor edits documents ‚Äî while keeping the file plain, open HTML that renders in any browser. Its core promise is the lossless round-trip: Ashokan never rewrites what you didn't edit. Everything outside <body> is preserved byte-for-byte; every element keeps its attributes; markup Ashokan doesn't model (scripts, SVG, widgets) is carried as protected islands that render but can't be mangled.

Working with AI agents (Claude Code, Codex, etc.)

Agents can also read the machine-oriented guide bundled with the app: Help > Agent Guide, or point them at /Applications/Ashokan.app/Contents/Resources/AGENTS.md.

Troubleshooting

  • Formatting buttons do nothing: you're in Suggest mode ‚Äî formatting changes can't yet be tracked, so they're blocked rather than applied silently. Toggle Suggest off.
