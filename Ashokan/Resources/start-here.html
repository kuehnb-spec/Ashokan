This is a live document — everything below is editable, and this file is yours (save it anywhere). Two minutes here shows you what Ashokan does.

1. This paragraph has suggestions waiting

Ashokan is a word processor for web pagesHTML and Markdown documents — the files AI agents write all day. Hover over the red or green text and a little ✓ / ✕ chip appears: accept or reject each suggestion with one click. Tracked changes here are proprietary magicstandard HTML that renders as redlines in any browser.

The review bar above has the bulk controls: Accept All ▾, Reject All ▾ (by author, even), and a Preview switch that shows the document as if everything were accepted — or rejected.

2. This sentence has a comment

Some feedback doesn't want to change the text — it wants to say something about it. Click the yellow highlight.

3. Make your own suggestions

Click Suggest in the top-right of the format bar, then type anywhere — your edits become green tracked insertions with your name on them, and deletions turn red instead of disappearing. Toggle Suggest off to edit normally.

4. Try these

  • Press ⌘/ — the source pane opens and you can watch the clean markup live
  • Type # then a space on an empty line — instant heading (Markdown shortcuts work everywhere)
  • Drag an image from Finder into this document — it embeds right into the file
  • Press ⌥⌘P — export a print-quality PDF
  • These checkboxes are real — click one

5. The part nobody else does

Ashokan never rewrites what you didn't edit. Styles, scripts, and markup it doesn't understand survive byte-for-byte — so the documents your AI agents produce stay theirs, and the edits stay yours. If you run Ollama, try Review → AI Review with Local Model: a model on your own machine redlines this document, and you judge its work with the same chips you just used. And Review → Add Agent Instructions embeds an invisible note that teaches Claude Code or Codex to send you redlines instead of rewrites.
